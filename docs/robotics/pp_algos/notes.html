1 Path planning

  • Computing a route from starting point to a goal point
  • assumptions
    • where you are and where u need to go and map of environment(perception)
    • behaviour of robot for command (math model)
    • obstacles and open space where robot can move
  • Find the set of commands required to move the robot to the desired position - path planning

2 Real time

  • something the happens without or negligible delay
  • in robotics,procces with 10 hz or higher is considered as realtime

3 Map representation

  • simplest map is a matrix of 0s and 1s
  • continous map - represented as mathematical function (X>0 or X<5,Y>0 or Y6)

4 path representation

  • series of interminent co-ordinates
  • entire vector field which gives a direction and magnitude
    • vector field is a function where you give x and y and it gives direction and magnitude to move in

5 list of algorithms

  • Artificial potential fields
  • Breadth first search
  • A*
  • RRT and RRT*
